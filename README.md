# fix-travis-ci

Repair annoying repetitive issues with Travis CI

## Getting Started
Install the module with: `npm install fix-travis-ci`

```js
var fix_travis_ci = require('fix-travis-ci');
fix_travis_ci.awesome(); // "awesome"
```
